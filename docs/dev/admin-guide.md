# 관리자 사용 가이드

이 문서는 노션 기반 견적서 관리 시스템의 관리자 기능 사용 방법을 안내합니다.

---

## 1. 관리자 로그인

### 접속 방법

브라우저에서 다음 URL로 이동합니다:

```
https://your-domain.com/admin/login
```

### 로그인 절차

1. 관리자 비밀번호 입력 필드에 환경변수 `ADMIN_PASSWORD`에 설정된 비밀번호를 입력합니다.
2. **로그인** 버튼을 클릭합니다.
3. 로그인 성공 시 견적서 목록 페이지(`/admin/invoices`)로 자동 이동합니다.

### 로그인 실패 시

- 비밀번호가 틀렸을 경우 오류 메시지가 표시됩니다.
- 비밀번호는 환경변수 `ADMIN_PASSWORD`에 설정된 값과 일치해야 합니다.

### 로그아웃

- 화면 우측 상단의 **로그아웃** 버튼을 클릭합니다.
- 세션이 삭제되고 로그인 페이지로 이동합니다.

> **주의**: 관리자 세션은 서버 메모리에 저장됩니다. 서버가 재시작되면 세션이 초기화되므로 다시 로그인해야 합니다. Vercel serverless 환경에서는 인스턴스별 메모리가 분리되므로 Redis/KV 스토리지 사용을 권장합니다.

---

## 2. 견적서 목록 조회

### 기본 목록 화면

로그인 후 `/admin/invoices` 페이지에서 노션 데이터베이스의 모든 견적서를 테이블 형태로 확인할 수 있습니다.

**테이블 컬럼 설명:**

| 컬럼 | 설명 |
|------|------|
| 견적서 번호 | 고유 식별 번호 (예: INV-001) |
| 클라이언트명 | 견적 대상 고객사/개인 이름 |
| 발행일 | 견적서 작성 날짜 |
| 유효기간 | 견적서 유효 만료 날짜 |
| 총액 | 견적서 합계 금액 |
| 상태 | 현재 처리 상태 (대기/승인/거절) |
| 링크 | 클라이언트 공유용 URL 및 복사 버튼 |

### 페이지네이션

- 기본으로 페이지당 10개 항목이 표시됩니다.
- 하단의 **이전** / **다음** 버튼 또는 페이지 번호를 클릭하여 이동합니다.

### 정렬

- 테이블 헤더(컬럼명)를 클릭하면 해당 컬럼 기준으로 오름차순/내림차순 정렬됩니다.
- 현재 정렬 컬럼에는 화살표 아이콘이 표시됩니다.

---

## 3. 검색 및 필터 사용법

### 키워드 검색

화면 상단의 검색창에 키워드를 입력합니다.

- **검색 가능 항목**: 클라이언트명, 견적서 번호
- 입력 즉시 실시간으로 결과가 필터링됩니다 (디바운싱 300ms 적용).
- 매칭된 텍스트는 노란색으로 하이라이팅되어 표시됩니다.

### 상태 필터

**상태** 드롭다운 메뉴에서 조회할 상태를 선택합니다.

| 옵션 | 설명 |
|------|------|
| 전체 | 모든 상태의 견적서 표시 |
| 대기 | 검토 중인 견적서 |
| 승인 | 클라이언트가 승인한 견적서 |
| 거절 | 클라이언트가 거절한 견적서 |

### 날짜 범위 필터

**시작일** ~ **종료일** 날짜 선택기를 이용해 특정 기간의 견적서만 조회합니다.

- 시작일만 입력하면 해당 날짜 이후 견적서가 모두 표시됩니다.
- 종료일만 입력하면 해당 날짜 이전 견적서가 모두 표시됩니다.

### 필터 초기화

검색창 옆의 **초기화** 버튼 또는 각 필터 선택 해제로 조건을 리셋합니다.

## 4. 링크 복사 및 공유

### 링크 복사

각 견적서 행의 **링크** 컬럼에서:

1. **복사** 버튼(클립보드 아이콘)을 클릭합니다.
2. 클립보드에 견적서 URL이 복사됩니다.
3. 화면 하단에 "링크가 복사되었습니다" 토스트 알림이 표시됩니다.
4. 버튼 아이콘이 체크 표시로 잠시 변경되어 복사 성공을 확인할 수 있습니다.

**생성되는 링크 형식:**
```
https://your-domain.com/invoice/[notionPageId]
```

### 새 탭에서 미리보기

링크 컬럼의 **외부 링크** 아이콘을 클릭하면 새 탭에서 클라이언트가 보게 될 견적서 페이지가 열립니다.

### 링크 공유

복사한 링크를 다음 방법으로 클라이언트에게 전달할 수 있습니다:

- 이메일 본문에 붙여넣기
- 카카오톡, 슬랙 등 메신저에 붙여넣기
- 문자 메시지 전송

---

## 5. 다크모드 설정

### 테마 전환 방법

화면 우측 상단의 **테마 토글** 버튼(해/달 아이콘)을 클릭합니다.

- **라이트 모드**: 밝은 배경의 기본 테마
- **다크 모드**: 어두운 배경의 테마로 야간 작업에 적합
- **시스템**: 운영체제의 다크모드 설정을 자동으로 따릅니다

### 설정 유지

선택한 테마는 브라우저의 로컬 스토리지에 저장되어 페이지 새로고침 및 재방문 시에도 유지됩니다.

---

## 6. 자주 묻는 질문 (FAQ)

**Q. 견적서가 목록에 표시되지 않습니다.**

A. 노션 데이터베이스와의 연동을 확인하세요. 환경변수 `NOTION_API_KEY`, `NOTION_INVOICES_DATABASE_ID`가 올바르게 설정되어 있는지 확인합니다. 노션 Integration에 해당 데이터베이스 접근 권한이 있어야 합니다.

**Q. 링크 복사가 작동하지 않습니다.**

A. 브라우저의 클립보드 권한을 확인하세요. HTTPS 환경에서만 Clipboard API가 정상 동작합니다. HTTP 환경(로컬 개발)에서는 execCommand 폴백이 사용됩니다.

**Q. 로그인 후 바로 로그아웃됩니다.**

A. Vercel serverless 환경에서는 인스턴스 재시작 시 세션이 초기화될 수 있습니다. 지속적인 세션 유지가 필요하다면 Redis/KV 기반 세션 스토리지 도입을 검토하세요.

**Q. 검색이 너무 느리게 반응합니다.**

A. 검색은 300ms 디바운싱이 적용되어 있습니다. 입력 완료 후 약 0.3초 뒤에 결과가 갱신됩니다.
